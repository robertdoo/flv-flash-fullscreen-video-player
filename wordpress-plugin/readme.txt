::: INSTALLATION :::

1. Copy the whole folder "flv-player-plugin" to "wp-content/plugins"
2. Copy the folder "flvplayer" to "wp-content/flvplayer"

::: ACTIVATION AND CONFIGURATION :::

1. Active the plug-in ("Plugins")
2. Configure the player ("Options > FLV Player")

::: USING THE FLV PLAYER :::

[flashvideo video=demo-video.flv /]
